# Healthcare-Fraud-Detection-using-ML
## 📌 Overview

The healthcare industry faces huge losses due to the mismanagement of insurance transactions. This repository implements a novel framework for analyzing health insurance data utilizing adaptive machine learning and deep learning techniques. Unlike traditional rule-based systems that require manual updates and yield high false-positive rates , this system introduces an evolving, real-time methodology designed to be inculcated directly into an enterprise's insurance claim management system.

##  Key Features
 **Level Detection:** Identifies anomalous behaviors by isolating the three key elements of the healthcare ecosystem: patients, providers (doctors/hospitals), and services.<br>
 **Adaptive Association Rule Engine:** Computes association scores based on the frequency of visits and prescriptions to filter high-frequency records for rule evaluation.<br>
 **Specialty-Specific Rule Generation:** Groups patient records by medical specialty and uses G-means clustering to compute support and confidence values.<br>
 **Time-Series Anomaly Transformer:** Converts patient service transactions into time-series sequences. The transformer uses both "normal" and "anomaly" attention mechanisms to learn typical data distributions and highlight deviating timestamps.<br>
